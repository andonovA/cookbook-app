# Feature Ideas for Cookbook App

## üî• High Priority Features

### 1. **User Profiles**
- Display user's name and avatar
- Show user's recipe count
- View all recipes by a specific user
- Profile page: `/users/[id]`

### 2. **Recipe Search & Filter**
- Search recipes by title/ingredients
- Filter by:
  - Cuisine type (Italian, Asian, Mexican, etc.)
  - Meal type (Breakfast, Lunch, Dinner, Dessert)
  - Cooking time
  - Difficulty level
- Sort by: newest, most popular, alphabetically

### 3. **Recipe Categories/Tags**
- Add tags to recipes (vegetarian, gluten-free, spicy, etc.)
- Category selection when creating recipe
- Filter by tags/categories on homepage

### 4. **Recipe Ratings & Reviews**
- 5-star rating system
- Written reviews/comments
- Average rating display
- Sort by highest rated

### 5. **Favorites/Save Recipes**
- "Save" or "Favorite" button on recipes
- My Favorites page showing saved recipes
- Quick access to saved recipes

### 10. **Recipe Collections**
- Create custom collections (e.g., "Weeknight Dinners", "Holiday Recipes")

- Share collections with others
